# Draft Script SOP

## Purpose

Convert the strongest angle into a script or content draft that can be reviewed or shipped.

## Required Inputs

- Angle board from `generate-angles`
- `campaignTheme`
- `callToAction`

## Procedure

1. Pick the strongest angle for the target audience.
2. Open with the hook immediately.
3. Build the script around one clear tension and one outcome.
4. Close with a CTA that matches the promise in the hook.

## Output Standard

- Produce a clear draft with headline, hook, and CTA.
- Keep the structure scannable and ready for review.
- The draft should read like content, not a prompt transcript.

## Failure Conditions

- The hook is weak or buried.
- The script rambles across multiple ideas.
- The CTA does not match the content promise.
