# Knowledge Library

This folder holds reusable operating knowledge for the platform.

Rules:
- Keep knowledge structured by domain and workflow, not as random notes.
- SOPs must describe repeatable operating behavior, not vague advice.
- If an agent depends on a SOP, that linkage should be explicit in code.
- Knowledge should stay retrieval-ready and versionable.

Current scope:
- `sops/`: workflow and agent operating procedures
